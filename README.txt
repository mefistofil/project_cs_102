Перед Вами интернет-магазин!
( все возможности и функции проиюллстрированы в папке screenshots, пожалуйста проверьте эти скриншоты. )
#использовался bootstrap по умолчанию 
Из админки возможно размещать новые заказы и создавать категории.
Он имеет расширенную админку, в которой можно посмотреть заказы, посмотреть заказы более подробно( что было куплено и за сколько ), экспорт заказов в CSV.
Присутствует система купонов. 
Настроена корзина покупателя, её статус отмечен в верхнем правом углу и обновляется, используя Ассинхронность задач (Celery, RabbitMQ).

celery -A myshop flower     -     #Команда, для запуска усправления ассинхронными задачами ( в отдельной консольке )
celery -A myshop worker -l info     -     #просмотр и создание очереди задач(в отдельной консольке)
http://localhost:5555/dashboard    -    #доска для просмотра активности задач

Интернет-магазин оснащён системой оплаты PayPall. При оплате учитываются все скидки по купонам и отправляются сообщения на почту о статусе заказа. 
Я использовал PayPall IPN, чтобы прослеживать статус транзакций в админке. Для того чтобы проверить работоспособность я воспользовался временным прокси : ngrok.
Стоит учесть, что при запуске ngrok есть лимит на 20 кликов в минуту, так что не стоит растраичвать необдуманно драгоценные обращения к серверу =)

Чтобы убедится в работоспособности оплаты я воспользовался песочницей, предоставленной PayPall. 
https://www.sandbox.paypal.com/us/cgi-bin/webscr?cmd=_history&dispatch=5885d80a13c0db1f8e263663d3faee8d83a0bf7db316a7beb1b14b43acd04037      - платёжка

Чтобы платёж прошёл, необходимо совершатиь оплату с почты example-bayer.com
https://developer.paypal.com/developer/accounts/     -     аккаунты 

Пользователям не предоставлена регистрация, однако каждый из них уникально работает только со своей карзиной покупок. Для того, чтобы добится такого результата
я воспользовался методами сессий, предоставленными Джаного : django.contrib.session
